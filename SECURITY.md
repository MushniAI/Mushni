# Security Policy

## Reporting Security Issues

Please do not open public GitHub issues for security vulnerabilities.

If you believe you have found a security issue related to Mushni, contact the maintainers through the official website:

https://mushni.com

## Responsible Disclosure

When reporting a security issue, include:

- A clear description of the issue
- Steps to reproduce it
- Impact assessment
- Any relevant screenshots or logs

Please avoid accessing, modifying, or sharing data that does not belong to you.

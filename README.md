# Mushni AI Creator Platform

Mushni is a commercial AI creator platform for building stories, comics, images, videos, interactive chat fiction, and repeatable creative production workflows.

This repository is the public documentation and SEO resource hub for Mushni. It does not contain the private platform source code. Instead, it explains the product ecosystem, creator workflows, onboarding guides, prompt patterns, and responsible AI publishing practices.

## Product Ecosystem

| Product | Focus | Link |
| --- | --- | --- |
| Mushni | Main AI creator platform and product hub | https://mushni.com |

## Safety and Compliance

Mushni is designed for responsible AI-assisted creation. Creators should review generated output before use, respect intellectual property rights, follow local laws, and comply with the rules of the platforms where content is published.

See [Safety and Compliance](docs/safety-and-compliance.md) for practical guidelines.

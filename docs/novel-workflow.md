# AI Novel Workflow with Mushni

This beginner workflow explains how to plan an AI-assisted novel project with Mushni-style creator tools.

## Goal

Create a structured novel concept that can be expanded into chapters, scenes, and publishable drafts.

## Steps

1. Write a one-paragraph project brief.
2. Create character cards for the protagonist, antagonist, and supporting characters.
3. Draft a three-act story outline.
4. Break the outline into chapters.
5. Generate a first draft chapter by chapter.
6. Review each chapter for continuity, tone, pacing, and safety.
7. Revise with specific instructions instead of regenerating everything.

## Recommended Mushni Path

- Use NovelAI.one for story planning, drafting, editing, reading, and publishing workflows.
- Use character templates to keep identity, voice, and relationship details stable.
- Move to ImgAI.one when the story is ready to become comic panels or visual scenes.

## Quality Checklist

- The protagonist has a clear goal.
- The conflict escalates over time.
- Character details stay consistent.
- Each chapter has a purpose.
- Human review is completed before publishing.

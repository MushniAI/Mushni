# Code of Conduct

Mushni aims to maintain a respectful, useful, and creator-friendly project space.

## Expected Behavior

- Be respectful and constructive.
- Stay on topic.
- Give clear feedback.
- Assume good faith when discussing ideas.
- Respect intellectual property and privacy.

## Unacceptable Behavior

- Harassment, abuse, or personal attacks
- Spam or unrelated promotion
- Sharing private information without permission
- Content that is illegal, exploitative, or intended to harm others
- Misleading claims about AI-generated work or project affiliation

## Reporting

If you see behavior that violates this code of conduct, please open a GitHub issue or contact the project maintainers through the official Mushni website.

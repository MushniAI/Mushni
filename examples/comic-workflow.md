# Example: AI Comic Workflow

This example shows a practical workflow for creating an AI-assisted comic concept.

## Goal

Turn a story idea into character references, scene beats, and storyboard-ready prompts.

## Steps

1. Write a short premise.
2. Create character cards with visual details.
3. Define the visual style guide.
4. Split the story into pages or episodes.
5. Create a shot list for each page.
6. Generate image prompts from the shot list.
7. Review image outputs for character consistency.
8. Revise prompts and regenerate only the panels that need changes.

## Recommended Files

- [Character Card](../templates/character-card.md)
- [Storyboard Shot List](../templates/storyboard-shot-list.md)

## Quality Checklist

- Character appearance is stable across panels.
- Background and lighting support the story.
- Panel actions are clear.
- Dialogue fits the available space.
- Final pages are reviewed before publication.

# Example: Interactive Chat Fiction Workflow

This example describes a workflow for planning interactive chat fiction or playable character scenes.

## Goal

Create a structured interactive story project with characters, scenes, branches, and publishable project metadata.

## Steps

1. Define the project premise.
2. Create character cards.
3. Define the opening scene.
4. Write the conversation style and boundaries.
5. Generate scene messages.
6. Add optional branches or player choices.
7. Test the player experience.
8. Review safety, tone, and continuity.
9. Publish or revise the project.

## Quality Checklist

- The player understands the opening situation.
- Character voice remains consistent.
- Branches do not contradict the story setup.
- Generated messages are reviewed before publishing.
- The project follows platform and local content rules.
